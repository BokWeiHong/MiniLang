## How to Compile and Run MiniLang

1. **Compile** the source code:
   - Using `g++`:
     ```bash
     g++ src/*.cpp -o minilang
     ```
   - Or using `clang++`:
     ```bash
     clang++ src/*.cpp -o minilang
     ```

2. **Run** the program with an input file:
   ```bash
   .\minilang input.lang
